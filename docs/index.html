TowerRush

一款零依赖、零构建的单页 HTML5 塔防游戏。24 种防御塔、7 类敌人、6 张地图、30 波进攻，配以程序化合成音效与在线排行榜回放。

Features

| 核心特性

🗼

24 种防御塔

每种 4–9 级成长，满级解锁专属 EX 形态：爆炸箭雨、连锁闪电、巡航导弹、灼日审判……

👾

多样敌人

快速、强壮、护盾、召唤者与最终首领，各有抗性与机制，逼你不断调整布局。

🎵

零文件音效

全部音效由 Web Audio API 实时合成，不依赖任何音频资源，加载即响。

🏆

排行榜 & 回放

LinuxDo 登录后分图上榜，可逐帧观看任意记录的完整操作回放。

📱

移动端适配

横屏、触控与高分屏自适应，手机上同样流畅可玩。

🧩

纯原生无依赖

没有框架、没有打包步骤。克隆下来双击 index.html 即可运行。

Get Started

| 本地运行

git clone https://github.com/Ooxygen7/TowerRush.git
cd TowerRush
# 直接双击 index.html，或：
python -m http.server 8080   # 访问 http://localhost:8080/

采用经典 <script>，file:// 直接打开即可运行，无需任何依赖或构建。完整数据表与部署指南见 README。
